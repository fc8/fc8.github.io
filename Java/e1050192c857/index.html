CLassLoader

alm0st
2024-04-21
Java
安全

JVM中的类加载器有：

  • Bootstrap ClassLoader（引导类加载器）
  • Extension ClassLoader（扩展类加载器）
  • App ClassLoader（系统类加载器）默认的类加载器，如果类加载时我们不指定类加载器的情况下，默认会使用AppClassLoader加载类
