OD使用总结

alm0st
2023-03-21
re
逆向
指令 快捷键 说明
Step Into F7 单步执行，进入
Step Over F8 单步执行，跳过
Restart Ctrl+F2 终止程序，重新载入
Go to Ctrl+G 跳转到指定地址
Run F9 运行，遇到断点停
Execute till return Ctrl+F9 执行函数代码，直到遇到RETN命令，用于跳出函数体
Execute till cursor F4 执行到光标所在位置
Comment ; 添加注释
Assemble 空格 编写汇编代码
Edit Data Ctrl+E 编辑数据
